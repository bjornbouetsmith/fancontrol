# fancontrol
Fan control/monitoring/alerting script for linux for DELL PowerEdge

## Works for
* PE R210II 
* PE R720

## Dependencies

* ipmitool
* sendmail

Probably best run with tmux or similar.

Could be enhanced with syslog capabilities.
